■名称
　JCBookmark

■概要
  ブックマークをPCローカルで管理するWindows用フリーソフトです。

■種別
　フリーソフト

■動作確認環境
　ＯＳ　　: Windows XP Home Edition SP3 (日本語版)
　ブラウザ: Google Chrome / Firefox / Internet Explorer 8 / Opera

■インストール方法
　ダウンロードしたアーカイブファイルを適当なフォルダに展開します。
　JCBookmarkフォルダができます。その中が本ソフトの本体です。

■使い方
　http://ztmsdf.appspot.com/jcbookmark/usage.html を参照してください。

■アンインストール方法
　ダウンロードしたアーカイブファイルとJCBookmarkフォルダを削除します。
　※プログラムの追加と削除に本ソフトはありません。
　※本ソフトからレジストリ書き込みはしません。
